Mobile

Jogue no Android

Minecraft Java no seu celular, com mods pré-configurados via PojavLauncher.

1
Instalar PojavLauncher

Baixe da Play Store ou GitHub. Gratuito e open-source.

2
Baixar o Pack Aether Mobile

Clique no botão abaixo para baixar o ZIP.

3
Extrair na pasta versions

Copie para Android/data/net.kdt.pojavlaunch/files/.minecraft/versions/

4
Copiar mods

Copie os .jar para .minecraft/mods/

5
Selecionar e Jogar

Selecione Aether-Client-1.20.1 no PojavLauncher.

Dúvidas

FAQ

O Aether Client é gratuito?

Sim, totalmente gratuito e open-source. O código está disponível no GitHub.

Precisa de conta original?

Não. Há suporte a conta offline (apelido livre) e conta Microsoft original. Com Microsoft, você acessa servidores como Hypixel.

O launcher instala mods automaticamente?

Sim. Ao iniciar o Minecraft, os mods são baixados do Modrinth com verificação de integridade SHA512.

Os cosméticos são visíveis para outros jogadores?

Visíveis para outros jogadores que também usam o Aether Client. Suporte multiplayer a outros clients está em desenvolvimento.

O Android usa Minecraft Java?

Sim, via PojavLauncher. Você joga a versão Java com mods Fabric no seu Android.

Como atualizar o launcher?

O launcher verifica atualizações automaticamente via GitHub Releases e notifica quando uma nova versão está disponível.
